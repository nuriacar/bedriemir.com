Python Kurulumu

 

Pyhton programlama dili, matematik uygulamarında çok kullanılmaktadır. Bunun nedeni, bu dilin tasarımında matematik öğelerine ağırlık verilmesi ve dilin çok sade fakat çok etkili bir sözyazımı (serntaks) ve içerik (semantik) yapısına sahip olmasıdır. Kullanacağımız birçok Bilgisayar Cebir Sistemi) (BCS) (Computer Algebra System = CAS) uygulama ortamlarının alt yapısında önemli yer tutmaktadır.

Çalışmalarımızda Python programlama diline özel bir ağırlık vereceğiz ve uygulamalarımız genişledikçe Python bilgimiz ve deneyimizi artacaktır.

Python çok yaygın bir programlama ortamıdır ve Web üzerinde büyük ölçüde serbestçe yararlanılabilecek öğrenim kaynakları bulunmaktadır. Python 3 üzerine yazılmış çok yararlı bir kitap, Wikipaedia tarafından yayınlanmıştır. Çok yararlı bir başka kaynak, Mark Pilgrim'in Dive into Python3 adlı Web üzerinde yayınlanmış kitabıdır. Bu olanaklardan yararlanılarak Python programlama dili üzerinde bilgi sahibi olmakta yarar bulunmaktadır. Python resmi sitesinin Python 3 üzerine eğitim veren kitabı çok yararlı olmaktadır. Yine Wikibooks tarafından yayınlanmış olan Python & Mat kitabı tam olarak tamamlanmamış olmasına karşın, çok yararlı bir bilgi kaynağıdır. Yine çok değerli bir kaynak Think Python Web de yayınlanmış bir ücretsiz kitaptır. Yine bir Wikibooks kitabı Python Programming çok yararlı olmaya devam eden bir kaynak kitaptır. A byte of Python bir başka yararlı kaynaktır. Wikibooks daha birçok Python öğrenim kaynağına linkler vermektedir. Daha başka birçok kaynak Web de bulunmaktadır.

Çalışmalarımızda kullanmak için kullandiğımız sistemde Python alt yapısını kurmakla işe başlamalıyız. Python resmi sitesi, https://www.python.org/ adresindedir. Buradan bilgisayarımız 32 bit Windows kullanıtyorsa, 32 bit en son sürüm Python 3 ü kuralım. Kurulum sırasında, Ptyhon.exe çalıştırılabilir dosyasını veriyolu (PATH) üzerine yerleştirme olanağını açık tutalım. Yerleşme sonucunda, c:\windows\System32\cmd.exe çalıştırılabilir dosyasını çalıştırarak komut satırına Python yazıp enter 'e basalım. Eğer Python veriyolu üzerine yerleşmiş ise, bir mesaj görüntelenerek yerleşmiş Python programının sürümü belirtilir. Eğer sistem bu komutu tanımıyorsa, Python.exe dosyası veriyolu üzerine el ile yerleştirilmelidir. Bu işlem de basitçe gerçekleştirilebilir. Web üzerinden http://patheditor2.codeplex.com/ adresinden Path Editor ücretsiz programını indirip kurun. Yönetici (Adminstartor) hakları ile çalıştırın, Python.exe dosyasını, ayrica, Java, gcc derleyicisi gibi, veriyolu üzerinde bulunması gerekli tüm dosyaları PathEditor yardımı ile veriyolu üzerine yerleştirebilirsiniz. Düzenleme bitince "save to registry" düğmesine basın. Yönetici hakları ile programı çalıştırdığınız için, veriyolu üzerinde yaptığınız değişiklikler Windows kayıt dosyası (Registry) 'ye kaydedilecektir.

Python çalıştırma ortamı olarak, PyScripter, Wing IDE 101, ve WinPython Tümleşik Program Geliştirme Ortamı (TPGO) (Integrated Development Environment = IDE) lerini öneriyoruz. Daha başka TPGO lar da bulunmaktadır. Örneğin Eclipse Python Pugg-in'i ile, Netbeans Python eklentisi ile, Visual Studio yine Python eklentisi ile çok iyi Python çalıştırabilir. Bu konuda Web üzerinden bilgi alarak bu ücretis programlar üzerinde Python çalıştırabilirsiniz. Üzerinde duracağımız ve biraz sonra üzerinde uygulama geliştireceğimiz Web üzerinde çalışan SAGE , CAS ortamı da Python programları çalıştırabiliyor, fakat şimdilik sadece Python 2 nin en son sürümü olan 2.7 yi destekleyebiliyor. Programlarımızda yazdırma kodunda küçük bir değişiklik ile SAGE ortamında da Python 2 çalıştırabileceğiz.

PyScripter'in en son sürümünü http://code.google.com/p/pyscripter/ adresinden siateminiz 32 bit veya 64 bit olması halinde en uygun yerleştirme dosyasını inderin ve Windows işletim sistemli bilgisayarınıza yerleştirin. 64 bitlik sistemler için, bu gün için en güncel yerleştirme programı, http://code.google.com/p/pyscripter/downloads/detail?name=PyScripter-v2.5.3-x64-Setup.exe dosyasıdır. Bu dosyayı yerleştirdikten sonra, seçekleri ile oynayarak, Pyton 3.4.1 sürümlü yorumlayıcıyı (interpreter) görecek şekilde kullanıma hazır hale getiriniz. Hazır olduktan sonra bir boş dosya açıp,

Print("Merhaba Dünya !");

yazarak çalıştırınız. Bu program sorunsuz çalışıp yorumlanabilirse, PyScripter IDE 'si kullanıma hazır demektir. Bir sorun çıkarsa, biraz daha uğraşmanız gerekebilir. Sonunda herşey güzel olacaktır.

Aynı şekilde, http://wingware.com/downloads/wingide-101 programını indirip kurun. Bu ücretsiz program, 32 ve 64 bitlik sistemler için aynı yerleştirme dosyasını kullanıyor. Aynen deneyip çalışmaya hazır hale getirin. Bu IDE hakkında kullanım yöntemi, http://wingware.com/doc/intro/tutorial sayfasında açıklanıyor.

Yine benzer şekilde, http://sourceforge.net/projects/winpython/files/WinPython_3.3/3.3.5.0/ sayfasından, winpython 32 veya 64 bitlik sürümünü indirin, aynı şekilde kurup çalıştırmaya hazırlayın.

Python çalıştırmak için, çok iyi bir uygulama ECLIPSE ortamıdır. Bu ortam üzerinde bilgisi olanlar, https://www.ics.uci.edu/~pattis/common/handouts/pythoneclipsejava/eclipsepython.html adresinde tarif edildiği şekilde kurup çalıştırabilirler.

Aynı şekilde, Microsoft Visual Studio'nun ücretsiz sürümü olan Visulal Studio Express de çok iyi bir Python çalıştırma ortamı sağlamaktadır. Bu konudaki bir video You Tube'den izlenebilir. Visual Studio Express http://www.microsoft.com/en-us/download/details.aspx?id=40787 den indirilebilir.Bu konuda deneyimi olanlar, bu programın yönergelerinden veya Iron Python tutorial sayfasından yararlanarak, Visual Studio Express de Python programlarını çalıştırabilecek ortamı hazırlayabilirler. Aynı şekilde, Netbeans de bu fonksiyonaliteyi sağlayabilir.

Bu> konuda, önceden deneyimi olmayanların bu IDE leri oluşturmak için uğraş vermeleri tamamı ile gereksiz olabilir. En basit kurulumu olan, kurulumları yukarıda açıklanmış PyScripter, Wing101, WinPython içindeki Sypyder light veya Spyder2 ortamları çalışmalarımızdaki örneklerin uygulanması için yeterli olacaktır.
