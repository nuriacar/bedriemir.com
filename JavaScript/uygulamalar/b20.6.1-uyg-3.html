3 Resimli JavaScript Rollover Etkisi
  • Sayfanın XHTML Kodları
  • Sayfanın CSS Stil Kodları
  • Sayfanın JavaScript Program Kodları

Bu Rollover Programının Özellikleri :

  • Bu JavaScript programı, farkedilmeyen kayboluş kriteryumunu karşılamaktadır. Yani eğer JavaScript yorumlayıcısı çalışmıyorsa, sadece JavaScript programının sağladığı resim değişimleri görülmeyecek, fakat bağlantılar çalışacak ve kullanıcıya aktarılacak bilgilerde bir eksiklik olmayacaktır.
  • Bu JavaScript programı, programların biribirlerinin okunmasını engellememesi prensibine uygun olarak yazılmıştır. Yani, XHTML, CSS ve JavaScript kodları, ayrı sayfalarda saklanmış ve XHTML sayfasına bağlantılıtir. XHTML sayfasında, satıriçi stil ve olay yöneticisi nitelikleri kullanılmamıştır. XHTML sayfasında sadece XHTML kodları bulunmakta, CSS kuralları ayrı CSS sayfasında, programlama işlemleri ayrı JavaScript sayfasında çözümlenmektedir.
  • Sadece W3C-DOM Düzey2 Çekirdek (CORE) yöntemleri kullanılmıştır.Zaman aşımına uğraması olasılığı yüksek, W3C-DOM Düzey2 HTML yöntemleri kullanılmamıştır.
  • Global Değişkenler Kullanılmamıştır.
  • Rollover resimleri, program belleğine indirilerek, kullanıcı etkisi ile resim değişimi hızlandırılmıştır.
